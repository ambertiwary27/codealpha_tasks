# 🌐 Language Translation Tool

A simple and responsive Language Translation Tool built using HTML, CSS, JavaScript and Node.js.

## 📌 Features

- 🌍 Translate text into multiple languages
- ⚡ Fast and responsive interface
- 🎨 Clean UI
- 📱 Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js

## 📂 Project Structure

```
translator-project/
│
├── index.html
├── style.css
├── script.js
├── package.json
├── server.js
└── README.md
```

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ambertiwary27/codealpha_tasks.git
```

Go to project folder

```bash
cd codealpha_tasks
```

Install dependencies

```bash
npm install
```

Run the project

```bash
node server.js
```
